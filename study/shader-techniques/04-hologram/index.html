04. Hologram (홀로그램)

작성일: 2026-01-13 구현 완료 난이도: ★★☆
DirectX 11 HLSL frac() Fresnel Glitch

Hologram - SF 영화처럼 반투명하게 빛나는 홀로그램 효과

이런 효과 본 적 있나요?

스타워즈에서 레아 공주가 "오비완 케노비, 당신만이 희망입니다" 하는 장면이나, 아이언맨이 허공에 UI 띄워놓고 만지작거리는 장면 본 적 있죠?

이런 SF스러운 홀로그램 효과를 만들어볼 거예요.

홀로그램의 특징

  • 반투명 - 뒤가 비쳐 보임
  • 스캔라인 - 수평선이 위로 흘러감
  • 외곽 글로우 - 가장자리가 더 밝음
  • 글리치 - 지직거리며 깜빡임

원리: 4가지 효과 조합

1. 스캔라인 - frac()으로 줄무늬 만들기

frac() 함수는 소수점 부분만 뽑아내요. 

frac(1.7) = 0.7
frac(2.3) = 0.3
frac(5.0) = 0.0

Y좌표에 밀도를 곱하면 0~1이 반복되면서 줄무늬가 생겨요. 

float scanline = frac(Y * density + Time);

2. 프레넬 - 외곽 글로우

이미 배운 프레넬! 가장자리가 더 밝게 빛나요. 

float fresnel = pow(1.0 - dot(N, V), power);

3. 글리치 - 노이즈로 깜빡임

노이즈로 불규칙하게 픽셀을 껐다 켜요. 

float glitch = noise(Y + Time);
if (glitch < threshold)
    discard;  // 픽셀 버리기

4. 알파 블렌딩 - 반투명

알파값을 조절해서 뒤가 비치게 만들어요.

홀로그램 = 4가지 조합

스캔라인 + 프레넬 + 글리치 + 알파

단계별 구현 과정

Step 1: 스캔라인만

Step 1: frac()으로 만든 스캔라인이 위로 흘러감

목적: frac()으로 줄무늬 만들고 애니메이션

// 월드 Y 좌표 사용
float y = input.WorldPos.y;

// frac으로 0~1 반복 + Time으로 스크롤
float scanValue = frac(y * ScanlineDensity + Time * ScanlineSpeed);

// step으로 줄무늬
float scanline = step(0.5, scanValue);

Step 2: 스캔라인 + 프레넬

Step 2: 외곽이 더 밝게 빛남!

목적: 가장자리 글로우 추가

// 프레넬 계산
float NdotV = dot(N, V);
float fresnel = pow(1.0 - saturate(NdotV), FresnelPower);

// 스캔라인 + 프레넬 결합
float intensity = fresnel + scanline * 0.3;
float3 color = HoloColor * intensity;

Step 3: + 글리치

Step 3: 지직거리는 글리치 효과!

목적: 불규칙한 깜빡임 추가

// 노이즈 생성
float glitchNoise = noise(y * 20.0 + Time * 10.0);

// 노이즈 값이 낮으면 픽셀 버리기
if (glitchNoise < GlitchIntensity)
{
    discard;
}

// 색수차 (chromatic aberration)
color.r += glitchOffset;
color.b -= glitchOffset;

Step 4: 최종 (반투명)

Step 4: 완성! 반투명 + 모든 효과

목적: 알파 블렌딩으로 진짜 홀로그램처럼

// 알파 계산 - 프레넬로 가장자리 더 불투명
float alpha = Alpha * (0.5 + fresnel * 0.5);

// 스캔라인 영역은 살짝 더 투명
alpha *= (0.8 + scanline * 0.2);

return float4(color, alpha);

단계별 요약

  • Step 1: 스캔라인 - frac()으로 줄무늬
  • Step 2: + 프레넬 - 외곽 글로우
  • Step 3: + 글리치 - 지직거림
  • Step 4: + 알파 - 반투명 완성

파라미터 조정

파라미터 설명 권장 범위 효과
ScanlineDensity 스캔라인 밀도 5 ~ 30 높을수록 줄무늬 촘촘
ScanlineSpeed 스캔라인 속도 1 ~ 5 높을수록 빠르게 흐름
FresnelPower 프레넬 강도 1 ~ 5 높을수록 외곽만 빛남
GlitchIntensity 글리치 강도 0 ~ 0.3 높을수록 많이 깜빡임
Alpha 전체 투명도 0.3 ~ 0.8 낮을수록 투명

학습 포인트

핵심 함수

  • frac(x): 소수점만 추출 → 반복 패턴
  • step(edge, x): 경계 기준 0 또는 1
  • pow(1-dot(N,V), p): 프레넬 효과
  • discard: 픽셀 버리기
