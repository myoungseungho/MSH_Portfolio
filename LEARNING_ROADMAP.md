# 쉐이더 테크닉 학습 로드맵

> DirectX 11 기반 8가지 쉐이더 테크닉 학습 계획

## 📚 학습 순서 (난이도순)

### ✅ 1단계: 기초 (추천: 이것부터!)

#### 1️⃣ Fresnel (프레넬 효과) ⭐
**난이도**: ★☆☆
**예상 소요**: 1일
**핵심 개념**: `dot(N, V)`, `pow()`, 시선-노말 벡터

**학습 목표**:
- 벡터 내적(dot product) 활용법
- 카메라 시선 벡터 계산
- Power 값으로 강도 조절

**구현 내용**:
- 구체 메시에 외곽 글로우 적용
- 프레넬 강도(power) 조절 UI
- 색상 변경 가능

**실전 활용**:
- 쉴드 효과
- 홀로그램 베이스
- 에너지 필드

---

#### 2️⃣ Toon Shading (툰 쉐이딩) ⭐
**난이도**: ★☆☆
**예상 소요**: 1-2일
**핵심 개념**: Lambert 조명, `step()`, `smoothstep()`

**학습 목표**:
- 기본 조명 계산 (Lambert)
- 계단식 음영 만들기
- 아웃라인 조합 (선택)

**구현 내용**:
- 방향성 조명 계산
- 2~3단계 음영 분리
- 하이라이트 추가

**실전 활용**:
- 셀 쉐이딩 캐릭터
- 카툰 렌더링 게임
- 만화풍 이펙트

---

#### 3️⃣ Dissolve (디졸브) ⭐
**난이도**: ★★☆
**예상 소요**: 1-2일
**핵심 개념**: `clip()`, 노이즈 텍스처, 타임라인

**학습 목표**:
- 텍스처 샘플링
- 알파 클리핑
- 시간 기반 애니메이션

**구현 내용**:
- 노이즈 텍스처 생성/로드
- Dissolve 진행도 조절 (0~1)
- 가장자리 글로우 효과

**실전 활용**:
- 캐릭터 소환/사망 효과
- 텔레포트
- 피격 이펙트

---

### ✅ 2단계: 중급 (기초 마스터 후)

#### 4️⃣ Hologram (홀로그램) ⭐⭐
**난이도**: ★★☆
**예상 소요**: 2일
**핵심 개념**: 효과 레이어링, `frac()`, UV 조작

**학습 목표**:
- 여러 효과 조합
- 스캔라인 효과
- 노이즈 깜빡임

**구현 내용**:
- Fresnel 베이스
- 수평 스캔라인 (`frac(uv.y * count + time)`)
- 랜덤 깜빡임
- 색상 변환 (청록색 계열)

**실전 활용**:
- SF 홀로그램
- UI 이펙트
- 텔레포트 이펙트

---

#### 5️⃣ Outline (외곽선) ⭐⭐
**난이도**: ★★☆
**예상 소요**: 2일
**핵심 개념**: 백페이스 렌더링, 노말 확장, 두 패스

**학습 목표**:
- 다중 렌더 패스
- 백페이스 컬링 제어
- 정점 쉐이더에서 메시 확장

**구현 내용**:
- Pass 1: 노말 방향으로 확장 + 백페이스만 렌더
- Pass 2: 일반 렌더링
- 외곽선 두께/색상 조절

**실전 활용**:
- 캐릭터 외곽선
- 선택된 오브젝트 하이라이트
- 만화풍 렌더링

---

#### 6️⃣ Screen Distortion (화면 왜곡) ⭐⭐
**난이도**: ★★☆
**예상 소요**: 2일
**핵심 개념**: UV 왜곡, 포스트 프로세싱

**학습 목표**:
- UV 오프셋 적용
- 렌더 타겟 활용
- 노이즈 기반 왜곡

**구현 내용**:
- 백버퍼 → 텍스처 렌더링
- 노이즈로 UV 왜곡
- 히트 웨이브 애니메이션

**실전 활용**:
- 히트 웨이브
- 공간 왜곡 효과
- 포탈 이펙트

---

### ✅ 3단계: 고급 (시간 투자 필요)

#### 7️⃣ Water/Wave (물 표면) ⭐⭐⭐
**난이도**: ★★★
**예상 소요**: 3-4일
**핵심 개념**: 사인파 중첩, 노말맵, 반사/굴절

**학습 목표**:
- 복수 사인파 합성
- 동적 노말 계산
- 환경 반사 시뮬레이션

**구현 내용**:
- 파도 높이맵 (여러 sin 파 중첩)
- 노말 계산 (수평/수직 미분)
- 프레넬 기반 반사/굴절 블렌딩
- UV 스크롤링

**실전 활용**:
- 실시간 물 표면
- 바다/강
- 마법진 이펙트

---

#### 8️⃣ Fire/Smoke (불/연기) ⭐⭐⭐
**난이도**: ★★★
**예상 소요**: 3-4일
**핵심 개념**: Perlin Noise, UV 스크롤, 알파 블렌딩

**학습 목표**:
- Perlin Noise 구현/활용
- 다중 UV 레이어
- 알파 블렌딩 최적화

**구현 내용**:
- Perlin Noise 텍스처 생성
- 2~3개 레이어 UV 스크롤 (다른 속도)
- 색상 그라디언트 (빨강→주황→노랑)
- 알파 페이드 (위로 갈수록 투명)

**실전 활용**:
- 불꽃 파티클
- 연기 이펙트
- 마법 스킬 효과
